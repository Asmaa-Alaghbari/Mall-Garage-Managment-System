Pentru a naviga între pagina de login și cea de sign up, vei avea nevoie de React Router
npm install react-router-dom


Vei avea nevoie de câteva pachete NuGet pentru a lucra cu PostgreSQL și pentru a implementa autentificarea JWT.
dotnet add package Npgsql.EntityFrameworkCore.PostgreSQL
dotnet add package Microsoft.EntityFrameworkCore.Design
dotnet add package Microsoft.AspNetCore.Authentication.JwtBearer

generezi migrările și să actualizezi baza de dat
dotnet ef migrations add InitialMigration
dotnet ef database update

pentru a face cereri HTTP din frontendul React
npm install axios

comanda openssl pentru a genera o cheie secretă
openssl rand -base64 32

// HashedPassword
dotnet add package BCrypt.Net-Next
